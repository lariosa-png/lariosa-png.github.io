Program Terror

Gameplay showcase

About

Program Terror is a First person horror game developed for my second year game design class in ESAT - Valencia , The initial concept of the project was to make a first person game without shooting and with a unique gameplay mechanic so since I'm a fan of the horror genre I decided to go for that and took games like SOMA for inspiration and as the unique gameplay mechanic i decided to use "programming" as a mechanic in the game in the form of programming doors to open them.

Development time: 3 months

Year of project completion: 2019

The game consists on programming the doors to open them while avoiding patrolling enemies that are on the lookout for the player.

Development

The game has been developed using unity engine and C#, The initial idea was always a game about hiding as is commonly used in horro games, development started with the player movements and mechanics these are the flashlight, hiding under tables and opening doors by writing the code shown on the screen. After the player I started working on the enemies, for these i used the unity pathfinding algorithm and points around the map that they go at random.
